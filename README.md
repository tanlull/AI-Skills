# agents

A collection of AI assistant skills, available globally via the [skills.sh](https://skills.sh/) registry.

## Available Skills

### MQL5 Assistant (`mql5`)

Expert MQL5 coding assistant for MetaTrader 5. It can write, fix, review, and explain MQL5 code (Expert Advisors, Indicators, Scripts).

#### Installation

You can install this skill globally to all your supported AI agents (e.g., Cline, Cursor, Antigravity, GitHub Copilot) directly from the registry:

```bash
npx skills add tanlull/agents --skill mql5
```

Alternatively, if you have this repository cloned locally, you can install the skill from the local directory:

```bash
npx skills add ./skills/mql5
```
